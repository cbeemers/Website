Maze!

This maze was created using a recursive backtracking algorithm that kept track of the path of visited nodes and went through each node's adjacents to remove a border. If all of the adjacent nodes were visited, it would backtrack in the path and continue going through nodes until all were visited. Solving was implemented in just about the same way but used a callback function for the animation as an argument that would specify if the goal was reached.
